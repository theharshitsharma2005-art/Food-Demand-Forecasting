# 🍽️ Food Demand Forecasting

## 📌 Project Overview

This project focuses on predicting the number of food orders placed at fulfillment centers using historical order data and machine learning techniques. Accurate demand forecasting helps businesses optimize inventory, reduce food waste, and improve operational efficiency.

---

# 🎯 Objectives

* Analyze historical food demand data.
* Merge multiple datasets for feature enrichment.
* Perform data preprocessing and exploratory data analysis (EDA).
* Train machine learning regression models.
* Predict future food demand.

---

# 📂 Dataset

This project uses three datasets:

* `train.csv` – Historical food demand records
* `meal_info.csv` – Information about meal categories and cuisines
* `fulfilment_center_info.csv` – Details about fulfillment centers

These datasets are included in this repository.
